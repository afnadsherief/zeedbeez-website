# Open Graph Images

The default Open Graph image is generated by
`src/app/opengraph-image.tsx` at `/opengraph-image` (1200 × 630 PNG).

It deliberately uses the approved brand token palette and copy so social
previews do not depend on an untracked or fabricated raster asset. An approved
campaign or product photograph may replace it in a future content release,
provided the metadata path and required 1200 × 630 format are preserved.
