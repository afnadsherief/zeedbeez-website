# Shader Guidelines

## Purpose
Define how custom shaders support the premium 3D experience without harming performance.

## Principles
- Use shaders only when they add real visual value.
- Keep shader logic reusable.
- Prefer simple, stable materials when performance matters.
- Test on desktop and mobile quality tiers.

## Targets
- Glass refraction
- Liquid-like honey motion
- Subtle glow and bloom support
- Premium translucency

## Rule
Every shader must have a documented purpose, fallback, and performance budget.
