# Architecture

## Principles
- Feature-first organization
- Reusable systems
- Strict TypeScript
- Performance-first
- Accessibility-first
- SEO-first

## Core Systems
- Scene manager
- Camera manager
- Animation manager
- Scroll manager
- Lighting manager
- Particle manager
- Asset loader
- Performance manager
- Shader library

## Implementation Rule
Do not duplicate scene logic across pages. Build shared systems and compose them per experience.
