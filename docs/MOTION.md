# MOTION.md

## Purpose
Define the motion language for the ZeedBeez Enterprise Platform.

## Motion Principles
- Motion must clarify, not distract.
- Motion should reinforce premium storytelling.
- Motion must support scroll-based narrative progression.
- Motion should remain calm, elegant, and purposeful.

## Motion Categories
### Structural Motion
- layout transitions
- section reveals
- chapter changes

### Product Motion
- bottle rotation
- ingredient drift
- honey flow
- lighting shifts

### Interface Motion
- hover states
- button response
- navigation transitions
- panel reveal

## Timing Rules
- Use restrained easing.
- Avoid overly playful motion.
- Match duration to storytelling importance.
- Use slower pacing for premium moments.

## Accessibility
Always support reduced motion mode with functional alternatives.
